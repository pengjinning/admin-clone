# Admin Clone

Chrome 插件，用于分析后台管理系统的界面结构，自动导出可编辑的 React + Ant Design 代码。

## 功能亮点

- 一键分析当前后台页面
- 遍历菜单并批量采集所有页面
- 自动生成 React + Ant Design 代码（后续会支持更多技术栈）
- 提供 Popup 与浏览器侧边栏双入口
- 自带构建、打包、同步、上传脚本
